    About unique username generator

    How does it work? We start with the most common words from a language. Break them down letter by letter and make a list of what follows what. Then generate a new unique word using the same probability of one letter following another. The result is (mostly) non-dictionary pronounceable nonsense words. For example, If "Q" is always followed by "U" in the english dictionary. Then "Q" will be followed by "U" in your english generated name.

    Optional Settings

    "Start With" "End With" and "Exclude" control the specific characters you want. For example, I normally exclude "l" and "i" because depending on the font they can be hard to distinguish. Another example would be to end with an "a" to give a more feminine sound.

    To add non-unique text to your unique name use the "before" and "after" settings. I often use "after" to add a domain extension (.com .io .xyz)

    "Language" is, well, the language that your generated username will sound like. Try latin for a scientific sounding nonsense word.

    Additional tips

    Check your new nonsense name uniqueness via Knowem using the button next to the generated username with the square and arrow. You can check the availability of that name across all the popular social media websites including Reddit, Blogger, YouTube, Twitter, LinkedIn, Facebook, Pinterest, Instagram and more.
